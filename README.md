# Nature Remo API(Python)

## Nature Remo API
- https://swagger.nature.global/
- https://local-swagger.nature.global/

## 動作環境
- Windows 10 
- Pyenv 2.3.1-12-g7457df41
- Python 3.7(Pyenv)
- Pipenv version 2022.6.7


## Pythonライブラリ
- requests
- json

## 実行方法
0. pipenv, pyenv pythonのインストールは完了していることを前提とする
1. pipenv install -e .
2. pipenv shell
3. python3 natureRemo.py
